Nome do Projeto: PFDA - Estudo de uso do Eclipse

Autor: Henrique Cabral Ferraz

Resumo: O projeto em questão tem por objetivo estudar e praticar o uso da IDE Eclipse 
através do clone de um repositório hospedado no github, criação de um arquivo em C 
responsável por calcular a média de 10 notas de um aluno, uso de licenças e arquivos xml.

Endereço eletrônico do repositório: https://github.com/Henrique-IFSP/PFDA.git
